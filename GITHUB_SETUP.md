## ğŸ’¡ **Git Commands Cheatsheet**

### **Daily Workflow:**

```bash
# Check status
git status

# Add changes
git add .

# Commit with message
git commit -m "feat: Add new feature"

# Push to GitHub
git push

# Pull latest changes
git pull
```

### **Branch Management:**

```bash
# Create new branch
git checkout -b feature/new-feature

# Switch branches
git checkout main

# Merge branch
git merge feature/new-feature
```

### **Undo Changes:**

```bash
# Undo last commit (keep changes)
git reset --soft HEAD~1

# Discard local changes
git checkout -- .

# View commit history
git log --oneline
```
