### ?? Advanced Perplexity AI Archiver
**Location:** `labs/experiments/perplexity_advanced_archiver.py`

**Description:** Comprehensive Perplexity AI chat history scraper leveraging the full AutomationCodex framework:

- **?? Mathematical Optimization:** Graph theory for crawling efficiency
- **?? Multi-Agent Coordination:** Parallel extraction using game theory
- **?? AI Template Inference:** Automatic pattern discovery and optimization  
- **? Quality Validation:** Statistical quality control and anomaly detection
- **?? Military-Grade Security:** AES-256 encrypted archives
- **?? Advanced Indexing:** Graph-based relationship analysis

**Quick Start:**
```bash
# Setup (first time only)
python labs/experiments/setup_perplexity_archiver.py

# Launch archiver
python labs/experiments/launch_perplexity_archiver.py

# Windows users
labs/experiments/run_perplexity_archiver.bat

# Comprehensive demo
python labs/experiments/demo_comprehensive.py
```

**Features:**
- Bulk extraction of all conversations, queries, responses, and files
- Intelligent parsing with AI-powered content structure detection
- Code block extraction with automatic language detection
- File reference tracking and catalog
- Conversation classification (Labs, Research, Pro, Standard)
- Self-healing pipelines with automatic error recovery
- Quality assurance with 7-dimensional scoring
- Full-text indexing and graph relationship analysis
